- Walk.py
Checking to be able to walk (Just up and down each foot)

Solving the inverse kinematics

- display_COG.py
Calculating the centor of gravity

- jacobian_test.py
Calculating the Jacobian

- COG_jacobian.py
Calculating the centor of gravity Jacobian
